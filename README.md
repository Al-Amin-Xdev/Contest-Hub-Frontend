Contest Hub Client Application

Contest Hub is a modern frontend web application built with React and Vite. It serves as the client-side interface for the Contest Hub platform, allowing users to discover contests, participate in them, make payments, submit work, and view results.

Features

React 19 with Vite for fast development

Firebase authentication (email, Google, etc.)

Stripe payment integration

Role-based user interface (user, creator, admin)

Interactive charts and analytics

Responsive UI using Tailwind CSS and DaisyUI

Form handling with React Hook Form

API state management using TanStack React Query

Smooth animations with Framer Motion

Map integration using Leaflet

Alerts and confirmations using SweetAlert2

Sliders and carousels using Swiper
